It is seamlessly integrated with Alexa providing live updates of content in fridge. I have also built PoC for insurance based company to classify voice based sentiments using machine learning. Academic Research Project

Skeleton based Recognition of Abnormal Human Activity in Bank ATM : Objective was to recognize abnormal human activity in ATM using skeleton data provided by Kinect sensor. It is achieved with two classification methodologies: Support Vector Machine and Dynamic Time Warping. Proposed solution achieved an accuracy of 95.24% using DTW (Dynamic Time Warping), 97.43% and 97.56% using SVM classification method when linear and polynomial kernels used respectively.
